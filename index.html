Remote management

Add and manage rclone remotes with the same wizard as rclone config.

File browser

Browse local and cloud paths, create folders, upload, download, and delete files.

Operations

Run copy, sync, move, check, and bisync with dry-run, filters, and bandwidth limits.

Live job monitor

Watch bytes, speed, and ETA update in real time over WebSocket.

Scheduled jobs

Cron-based automated backups with include/exclude patterns.
